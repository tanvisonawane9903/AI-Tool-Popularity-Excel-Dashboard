# AI Tool Popularity Dashboard (Excel)

An interactive Excel dashboard analyzing usage trends and market adoption of various AI tools, built to turn raw usage data into a clear, filterable view of how different tools are trending.

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Power Query (data transformation and cleaning)
- Slicers and dynamic filters
- Chart-based data visualization

## What the Dashboard Shows
- Usage trends across multiple AI tools over time
- Market adoption patterns and popularity comparisons
- Interactive filtering by tool, category, and time period

## How It Was Built
1. Raw usage data was imported and cleaned using **Power Query**.
2. **Pivot Tables** were used to summarize and structure the data for analysis.
3. Charts and visual elements were layered on top to highlight trends at a glance.
4. **Slicers** were added so the dashboard can be filtered interactively without touching the underlying data.

## Files
- `AI_Tool_Popularity_Dashboard.xlsx` — the full interactive dashboard (open in Excel to use the slicers and filters)

## How to Use
1. Download `AI_Tool_Popularity_Dashboard.xlsx`
2. Open it in Microsoft Excel (Pivot Tables and slicers require Excel, not Google Sheets, to work correctly)
3. Use the slicers on the dashboard sheet to filter by tool, category, or time period
